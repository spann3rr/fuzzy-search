# Getting Started with Vite

This project was deployed by VITE https://vitejs.dev/.

## Available Scripts

### Firsts install the packages 
### `npm install`

In the project directory, you can run:

### `npm run dev`
### `http://localhost:5173/`

To Build the project:

### `npm run build`
